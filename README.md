# JavaProgramm-ng-library-automation
A library web system built using Java programming
Bu proje, **Spring Boot**, **Java** ve **MySQL** kullanarak geliştirilmiş bir basit kütüphane yönetim sistemidir.
Kullanıcılar kitapları ve yazarları web sayfasına girerek veritabanına kaydederler.Aynı zamanda web sayfasında da bu kitap tablosu görünür.

## Özellikler

- Kitap ekleme, silme, listeleme
- Kullanıcı yönetimi
- Kitapların müsaitlik durumu

  ## Teknolojiler

- Java 17
- Spring Boot 3.x
- MySQL 8.x
- Maven
- RESTful API

  ## Calıstırma
  -Java kodunu calıştırın.
  - Daha sonra tarayıcıya şu adresi girin : http://localhost:8080/index.html
